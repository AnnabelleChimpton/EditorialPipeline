# FIRST-PASS EVENT CARD RUBRIC

This rubric is for first-pass event cards — fast triage, not deep analysis.

## Goal
Produce a compact event card that establishes what happened, who's reporting it, and how confident we are. Speed over depth. The card should be good enough to decide whether to invest a deep-dive pass.

## Minimum acceptance bar
An event card is acceptable only if it includes:
- 3-5 sources from different outlets (not rewrites of the same wire)
- At least 1 wire service (AP, Reuters, AFP) or official statement when available
- Timestamp on every major claim
- Confidence level (confirmed/developing/unverified) that matches the source corroboration level
- Urgency tier (critical/high/standard) with clear rationale
- Actors named with official titles on first reference
- Conflicting claims section filled (even if "no significant conflicting reports at this time")
- Open questions section filled
- Context needed section filled

## Confidence assignment rules

### Confirmed
Use when:
- 2+ independent sources (different outlets, not rewrites of the same wire report)
- At least one wire service or official statement
- Core facts are consistent across sources

### Developing
Use when:
- Single credible source (named reporter at established outlet, wire service, official statement)
- Facts may shift as more information emerges
- No contradicting reports, but insufficient corroboration

### Unverified
Use when:
- Social media origin, single unconfirmed source, or anonymous claim
- Conflicting reports from different parties
- Source has a track record of inaccuracy or clear partisan framing

## Anti-slop rules
Do not pass an event card as acceptable if it:
- Editorializes in the triage phase ("this shocking attack," "in a dramatic escalation")
- Uses "reportedly" without attributing to a specific outlet or source
- Cites unnamed sources without noting that they are unnamed
- Presents one party's framing as neutral fact
- Uses "tensions" or "the situation" without specifying between whom or what situation
- Omits the timestamp of the event being reported
- Has a confidence level that doesn't match the actual source corroboration (e.g., "confirmed" with only one social media post)

## Speed rules
- Max 3 web searches per card. If the first 2 searches establish the core facts with corroboration, stop.
- Don't chase historical context at this stage — that's Lane 3's job.
- Focus on who, what, when, where, and according to whom. Why and what it means come later.
