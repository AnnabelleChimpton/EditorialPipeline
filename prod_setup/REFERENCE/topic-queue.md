# TOPIC QUEUE

Use this file to separate topic sourcing from dossier production.

## Status meanings
- `queued` — ready for a first-pass dossier run
- `researching` — currently assigned or in progress
- `strong` — first-pass dossier says this is worth future deepening
- `maybe` — interesting but needs better sourcing or a sharper angle
- `weak` — not worth more time unless new evidence appears
- `deepen-next` — promoted for a later deeper research pass

## Queue fields
- Topic
- Hook
- Category
- Era
- Priority
- Sourceability
- Status
- Notes
- Dossier
